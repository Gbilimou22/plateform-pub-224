# PubWatch Pro

Plateforme de récompenses publicitaires - Gagnez de l'argent en regardant des publicités.

## 🚀 Fonctionnalités

- 👤 Inscription/Connexion sécurisée
- 📺 Visionnage de vidéos rémunéré (100 000 FCFA/vidéo)
- 💰 Système de retrait Mobile Money
- 🤝 Parrainage (10% des gains des filleuls)
- 🏆 Système de niveaux et badges
- 📅 Bonus de connexion quotidien
- 🎯 Défis quotidiens
- 🥇 Classement des meilleurs gagneurs
- 🎬 Intégration YouTube automatique
- 📊 Panneau d'administration complet

## 📋 Prérequis

- PHP 7.4+
- MySQL 5.7+
- Apache/Nginx
- Composer (optionnel)
